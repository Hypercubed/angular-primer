AngularJs module for creating an interactive SVG primer visualization. This library currently depends on d3 and requires angularjs 1.3.0-rc or higher.

Introduction

angular-primer is a set of angualrjs directives that can be easily added to existing angularjs applications to visualize genomic features. While originally designed for the visualization of DNA/RNA primers, the angular-primer directives composable elements can be used together to create a simple, or complex, SVG visualizations of any strand based genomic features. A visulaization consists of one or more primer-track elements. Each primer-track element can contain any number of primer-scale, primer-label, and primer-feature elements. A primer-feature in turn can contain any number of primer-label and primer-feature-shape elements. Since the view is built using SVG elements you have can add any additional SVG attributes, style with CSS, or extend behavior with other angularjs directives.
