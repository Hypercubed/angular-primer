primerFeatureShape
directive in module angularprimer

Description

Creates a shape in a feature

Usage

as element:
<primer-feature-shape
       [primer-feature-shape="{string}"]
       [height="{string}"]>
</primer-feature-shape>
as attribute
<path primer-feature-shape="{string}"
     [height="{string}"]>
   ...
</path>

Parameters

ParamTypeDetails
primerFeatureShape
(optional)
string

The feature shape. Can be arrow-right, arrow-left, or rect.

(default: 'rect')

height
(optional)
string

The shape hieght. Defaults to height if the feature.

(default: 'feature.height()')
