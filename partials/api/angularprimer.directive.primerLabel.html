primerLabel
directive in module angularprimer

Description

Adds a label to a feature or a track.

Usage

as element:
<primer-label
       [orient="{string}"]
       [anchor="{string}"]>
</primer-label>
as attribute
<g primer-label
     [orient="{string}"]
     [anchor="{string}"]>
   ...
</g>

Parameters

ParamTypeDetails
orient
(optional)
string

The label vertical orientation. Can be middle, top, or bottom.

(default: 'middle')

anchor
(optional)
string

The label hrizontal anchor. Can be middle, start, end.

(default: 'middle')
