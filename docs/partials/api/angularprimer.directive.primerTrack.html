primerTrack
directive in module angularprimer

Description

Creates a container used to group features, scales, and labels.

Usage

as attribute
<g primer-track
     [sequence="{string}"]
     [sequence-length="{number|expression}"]
     [start="{number|expression}"]
     [width="{number|expression}"]
     [height="{number|expression}"]>
   ...
</g>

Parameters

ParamTypeDetails
sequence
(optional)
string

The sequence

sequenceLength
(optional)
numberexpression

The sequence length used to scale the track

(default: sequence.length || 100)

start
(optional)
numberexpression

The sequence starting value (default = 0)

(default: 1)

width
(optional)
numberexpression

The track width, in pixels

(default: 500)

height
(optional)
numberexpression

The track height, in pixels

(default: 10)
