primerScale
directive in module angularprimer

Description

Adds a axis to a track.

Usage

as attribute
<g primer-scale
     [orient="{string}"]
     [ticks="{number}"]
     [outer-tick-size="{number}"]
     [inner-tick-size="{number}"]
     [tick-padding="{number}"]>
   ...
</g>

Parameters

ParamTypeDetails
orient
(optional)
string

The axis orientation. Can be top, or bottom.

(default: 'top')

ticks
(optional)
number

Approximate nuber of tick markers to display along axis.

(default: 5)

outerTickSize
(optional)
number

The outer tick size, in pixels.

(default: 6)

innerTickSize
(optional)
number

The inner tick size, in pixels.

(default: 6)

tickPadding
(optional)
number

The tick padding, in pixels.

(default: 3)
