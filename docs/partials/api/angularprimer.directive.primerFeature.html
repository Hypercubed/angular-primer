primerFeature
directive in module angularprimer

Description

Creates a feature on a track

Usage

as attribute
<g primer-feature
     start="{number|expression}"
     end="{number|expression}"
     [height="{number|expression}"]
     [direction="{string}"]>
   ...
</g>

Parameters

ParamTypeDetails
startnumberexpression

The feature starting position

endnumberexpression

The feature ending position

height
(optional)
numberexpression

The feature height

(default: track.height)

direction
(optional)
string

The feature direction. Can be left, right, or none.

(default: 'none')
