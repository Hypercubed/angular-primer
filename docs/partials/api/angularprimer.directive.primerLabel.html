primerLabel
directive in module angularprimer

Description

Adds a label to a feature or a track.

Usage

as attribute
<g primer-label
     [orient="{string}"]>
   ...
</g>

Parameters

ParamTypeDetails
orient
(optional)
string

The label orientation. Can be middle, left, right, top, or bottom.

(default: 'middle')
